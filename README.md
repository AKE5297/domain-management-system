# 域名管理系统

一个功能齐全的域名管理系统，帮助用户轻松管理和监控所有域名资产，跟踪到期时间，接收续费提醒，确保域名安全。

## ✨ 功能特性

- 🔐 **用户认证系统**：安全的登录认证机制
- 📋 **域名管理**：添加、编辑、删除和查看域名详细信息
- 📊 **数据分析面板**：直观的图表展示域名状态分布和到期趋势
- 📅 **续费日历**：年/月/周多视图日历，清晰展示域名到期时间
- ⏰ **到期提醒**：自定义的域名和SSL证书到期提醒系统
- 🎨 **响应式设计**：支持桌面和移动设备，提供浅色/深色主题切换
- 📧 **邮件服务配置**：自定义SMTP服务器设置，接收系统通知
- 🛠️ **偏好设置**：个性化系统配置和默认视图设置

## 🚀 快速开始

### 本地开发

1. 确保您已安装 Node.js (>=16.0.0) 和 pnpm

2. 克隆项目仓库

3. 安装依赖
   ```bash
   pnpm install
   ```

4. 启动开发服务器
   ```bash
   pnpm dev
   ```

5. 访问 http://localhost:3000 查看应用

### 构建部署

#### Cloudflare Pages 部署

1. 确保您已创建了 GitHub 或 GitLab 仓库并推送到远程
2. 登录 Cloudflare 控制台，导航到 Pages
3. 连接您的仓库，配置构建设置：
   - 构建命令: `npm install -g pnpm && pnpm install --no-frozen-lockfile && pnpm run build`
   - 构建输出目录: `dist/static`
   - 环境变量: `NODE_ENV=production`
4. 点击"部署站点"完成部署

#### Docker 部署

1. 确保您已安装 Docker 和 Docker Compose
2. 运行以下命令构建和启动容器
   ```bash
   docker-compose up -d
   ```
3. 访问 http://localhost:3000 查看应用

## 📝 使用说明

### 登录系统

系统提供默认管理员账户：
- 用户名: admin
- 密码: admin123

首次登录后，建议修改密码以保障安全。

### 域名管理

1. 点击"添加域名"按钮输入域名信息
2. 系统会自动计算域名状态（活跃、即将到期、已过期）
3. 您可以随时编辑或删除域名信息
4. 系统支持ICP备案信息管理

### 数据分析

1. 控制面板提供域名总数、活跃域名、即将到期和已过期域名的统计
2. 饼图展示域名状态分布
3. 柱状图展示月度到期趋势
4. 系统提供域名活跃度和过期风险评估

### 邮件通知设置

1. 在"系统设置" -> "邮箱服务"中配置SMTP服务器
2. 启用域名到期提醒和SSL证书到期提醒
3. 自定义提醒时间点（30天、15天、7天、3天、1天）

## 🔧 技术栈

- **前端框架**: React 18+
- **编程语言**: TypeScript
- **构建工具**: Vite
- **样式框架**: Tailwind CSS
- **图标库**: Font Awesome, lucide-react
- **图表组件**: Recharts
- **路由管理**: React Router
- **表单验证**: Zod
- **状态管理**: Context API + LocalStorage
- **部署平台**: Cloudflare Pages, Docker

## 📋 注意事项

- 系统使用LocalStorage存储数据，请注意数据备份
- 邮件通知功能需要正确配置SMTP服务器
- 默认情况下，系统会在域名到期前发送提醒通知
- 在公共设备上使用后，请及时退出登录
